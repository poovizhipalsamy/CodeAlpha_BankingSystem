Banking System in C++
#Features

* Create Account
* Deposit Money
* Withdraw Money
* View Account Details
* Transaction History

#Technologies Used

* C++
* Object-Oriented Programming (OOP)
* Vectors
* Strings

#How It Works

1. Create a bank account by entering customer details.
2. Deposit money into the account.
3. Withdraw money from the account.
4. View account information and current balance.
5. Check transaction history.

#Learning Outcomes

* Classes and Objects
* Functions
* User Input Handling
* Basic Banking Operations
